# Contributing

Thanks for your interest in improving Interactive Skill Tree Webpage.

## Development Workflow

1. Install dependencies with `pnpm install`.
2. Start the local app with `pnpm dev`.
3. Keep changes focused and include tests when touching logic in `src/app/lib`.
4. Run `pnpm test:run` and `pnpm build` before opening a pull request.

## Code Style

- Prefer existing components and data structures before adding new abstractions.
- Keep user-facing copy concise and consistent with the current Chinese product-development workflow language.
- Keep generated plans exportable as Markdown and JSON.

## Documentation

When behavior changes, update the relevant documentation under `docs/`.
